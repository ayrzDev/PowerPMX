## What is this?
PowerPMX is a highly customisable server software for Minecraft: Bedrock Edition, built from scratch in PHP, with over 10 years of history. It is based on PocketMine-MP, offering enhanced features and capabilities.

If you're looking to create a Minecraft: Bedrock server with **custom functionality**, look no further.
